# Figures
These PNGs are **sample outputs** produced by a smoke-test run of
`notebooks/07_SPICE_Validation.ipynb` and
`notebooks/08_Biomedical_Case_Study.ipynb` in analytical-oracle mode on
a small N (32 conversions, 500 MC trials, 4³ grid). On a full
`make reproduce` run with N=1000 and 10k MC trials they will be
overwritten with higher-resolution versions. They are committed here
so reviewers can see the expected shape of the outputs at a glance.
