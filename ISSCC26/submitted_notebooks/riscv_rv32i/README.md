# RISC-V RV32I Open Source Design

## 📌 Project Overview

This project implements a RISC-V RV32I processor and demonstrates its physical design using the OpenLane flow.

## 🛠️ Tools Used

* Verilog (RTL Design)
* OpenLane (Physical Design)
* Jupyter Notebook (Documentation)

## ⚙️ Design Flow

The following steps were performed:

1. Synthesis
2. Floorplanning
3. Placement
4. Clock Tree Synthesis (CTS)
5. Routing
